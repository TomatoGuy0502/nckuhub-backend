
# NCKUHUB BACKEND

NCKUHUB的後端，使用Nest.js開發


## Run Locally

Clone the project

```bash
  git clone https://github.com/NCKUKIWI/nckuhub-backend.git
```

Go to the project directory

```bash
  cd nckuhub-backend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start:dev
```


## Running Tests

To run unit tests, run the following command

```bash
  yarn test
```
